Information on the version of The Sentinel analysed here

Firebird released The Sentinel for the BBC Micro in 1986. It was available on both cassette and disc, though the game code is identical on both media. The version analysed on this site is from the Play It Again Sam 6 compilation, which is exactly the same game as the original release but with the copy protection removed.
