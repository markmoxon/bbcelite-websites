How to download and play the Elite Compendium on the BBC Master

You can run the Elite Compendium on a BBC Master 128 with either MOS 3.20 or MOS 3.50. Here are your options:

  • Play the Elite Compendium in your browser. This link will work for all but the 6502 Second Processor version; if you want to play the latter, you'll need this link.
  • Download the Elite Compendium as a disc image. This will run in an emulator, or on a real BBC Master 128 using a device like a Gotek.

To load the Elite Compendium, insert the disc image and press SHIFT-D-BREAK (or SHIFT-D-F12 on most emulators). The menu system will show you all the available options, and you can either read more information by highlighting an option and pressing the left or right arrow, or you can choose which option to run by pressing RETURN.

All options work on a standard BBC Master 128 except for 6502 Second Processor Elite, which needs a BBC Master Turbo (i.e. a BBC Master 128 with an internal 65C102 Second Processor) or a BBC Master 128 with an external 6502 Second Processor.

Commander files for the game are on drive 0, while universe files for the Universe Editor are on drive 2. When playing Teletext Elite or BBC Micro Elite, ensure that the disc is inserted when launching or docking, so the game can load the correct files (just as with the original BBC Micro disc version).

Note that the Elite Compendium only works on the BBC Master, and will not work on the BBC Micro.

Release history
---------------

The Elite Compendium has had the following releases:

  • 2024-01-11 - Initial release
  • 2024-01-12 - Fixed the Data on System bug in the 6502 Second Processor version
  • 2024-05-04 - Fixed an issue with corrupted ship rotation on the BBC Master version
  • 2024-05-15 - Fixed the Moray bug from the original BBC Master and 6502 Second Processor versions, and fixed an issue with Thargoids corrupting the screen in the BBC Master version
  • 2024-05-17 - Fixed an issue with Thargoids corrupting the screen in the 6502 Second Processor version

You can check the release for a given disc image by loading the disc and typing *TYPE README to display the credits. The build date is at the end.
