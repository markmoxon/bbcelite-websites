The easiest way to play Revs is online. The "Play online" links below will run the game right here in your browser, using JSBeeb.

You might find the Micro User Guide to Playing Revs useful.

Note that modern keyboards don't have the same range of keys as the BBC Micro, or even the same layout. In particular, BBC keys like f0 and COPY don't tend to exist anymore, so you may have to check the documentation to see which keys you should press instead. In JSBeeb you should press f10 instead of f0, for example.

If you want to play the game on a real vintage computer or in an emulator that you've installed on your modern machine, then there are links below to the relevant disc images. To play Revs with an SSD disc image, load the disc image into drive 0 and press SHIFT-BREAK.
